# Circuit Explorers: Hands-On Electronics with Raspberry Pi Pico 2W

Welcome to Circuit Explorers! This course teaches you to build real electronic circuits and program them using MicroPython on the Raspberry Pi Pico 2W.

## What You Need

- Raspberry Pi Pico 2W
- USB cable (micro-USB)
- Freenove Basic Starter Kit (FNK0064)
- Computer with [Thonny IDE](https://thonny.org) installed

### Additional Components (your instructor will let you know when these are needed)

| Item | Used Starting In |
|------|-----------------|
| SG90 Micro Servo | Module 9 |
| SSD1306 0.96" OLED Display (I2C) | Module 12 |
| HC-SR04 Ultrasonic Sensor | Module 13 |
| HC-SR501 PIR Motion Sensor | Module 14 |
| WS2812 NeoPixel LED Strip (8 LEDs) | Module 18 |

## How to Use This Repository

Each module has:
- **`lesson.ipynb`** — Your lesson notebook. Open it in Thonny or Jupyter. Follow along with your instructor to fill in the blanks.
- **`handout.md`** — A printable cheat sheet for quick reference.

The `reference/` folder has pin diagrams and a MicroPython cheat sheet you can use anytime.

## Getting Started

1. Install [Thonny](https://thonny.org)
2. Plug in your Pico 2W (hold BOOTSEL while plugging in the first time)
3. In Thonny, go to **Run > Configure Interpreter** and select **MicroPython (Raspberry Pi Pico)**
4. Open `student/modules/01-welcome-to-pico/lesson.ipynb` and follow along!
